# FinSight AI 💰

FinSight AI is an AI-powered personal finance management application that helps users track expenses, manage budgets, and gain smart financial insights.

## 🚀 Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Google Gemini AI

## ✨ Features

- User Authentication
- Expense Management
- AI Financial Insights
- Budget Tracking
- Secure REST APIs
